# AGENTS.md

This repository uses a branch-based template model.

## What This Repo Is

- `main` is the navigation branch.
- Template branches contain the real project skeletons and workflow defaults.

## Read Order

1. `README.md`
2. `CLAUDE.md`
3. `.planning/config.json`

## Which Branch Does What

- `main` — choose a template branch
- `template/gsd-default` — default path is `GSD -> execute-phase`
- `template/claude-codex-default` — default path is `Claude/GSD planning -> Codex bridge execution`
- `template/standalone-readiness` — experimental automation sandbox

## Detailed Docs

- `CLAUDE.md` — workflow constitution and execution posture
- `ARCHITECTURE.md` — branch and system design
- `RELIABILITY.md` — self-test, runtime artifacts, and operational limits

## Rule

Do not treat `main` as the template to build from. Pick a template branch first.
